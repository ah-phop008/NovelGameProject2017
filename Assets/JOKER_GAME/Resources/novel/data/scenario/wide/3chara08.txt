;タイトル　２学期の学校と体育祭

;背景　校門の前

#

学校に行っていたからか、２学期の始まりも苦ではない。[p]

自分で手入れした花壇は、見ていてとても気持ちいい。[p]

今までは全く意識していなかったけど、[r]
去年から雪雫は花壇を守ってきたのだろう。[p]

;雪雫表示

#雪雫
「おはよう。今日はぎりぎりじゃないのね」[p]

#蛍
「夏休みにちゃんと訓練してたからかもしれない」[p]

#雪雫
「訓練？なにかの部隊にでも入っているの？」[p]

#
突拍子もないことを言われた。[p]

#蛍
「違うよ、夏休みに早く学校に行く日が多かったから」[p]

#雪雫
「当たり前なはずだけど確かにあなたにしてみたらすごいことね」[p]

#
うすうす思っていたが、よく光みたいにオレを[r]
バカにしてくるよな。[p]

#雪雫
「でも、朝早いのに来てもらっちゃって[r]
無理をさせてしまったかしら」[p]

#蛍
「いや、こちらが感謝することたくさんあるから」[p]

#雪雫
「そう？じゃあまたお願いしようかしら。ありがとう」[p]

#担任
「席についてー」[p]

#雪雫
「じゃあまたお昼の時間に」[p]

#
朝は起きられるようになったが、授業はやっぱりだめだ。[p]

あー、はやく屋上で寝たい…[p]

:背景　校庭

２学期が始まったということは、体育祭が近づいてきた[r]
ということだ。[p]

行事はあまり好きではないが、運動なので一応参加はしている。[p]

だが、体育祭までの体育の授業時間が体育祭に向けた練習になる。[r]
正直面倒くさいけど。[p]

;光表示

#光
「今年は赤組で蛍と同じチームだな」[p]

#蛍
「別にどうでもいいけど」[p]

#光
「とか言ってるわりに、最近は全然授業さぼってないじゃないか」[p]

#蛍
「ただそういう気分なだけ」[p]

#光
「それにしても、ムカデ競争って難しいな」[p]

#蛍
「距離が近いし、転ぶし、暑いし、ムカデ競争って嫌いなんだよね」[p]

#
せめて大繩とかの方がましだ。[p]

;回想シーンなので、背景を光っぽくする

;雪雫表示

;背景　夜の神社

#雪雫
「走れる足があるのに走りたくないって言っている人を見て、[r]
嫌になって罪のないその人たちを恨んだりもしたわ」[p]

「今では現実を受け入れられるようになったけれど。[r]
私、最低だから誰かに当たっていたのよ」[p]

;回想シーン終わり

;背景　校庭

#
この言葉がすごく印象に残っている。[p]

そのせいかわからないけど、確かに最近は授業をさぼっていない。[p]

偉いな。凄い進歩だぞ、オレ。[p]

雪雫は、今日も制服より汚れていないジャージを着て[r]
木の下で見学をしている。[p]

#蛍
「暑くない？」[p]

#雪雫
「動いているあなたよりは暑くないわ」[p]

#
そうかもしれないが、この気温じゃ日陰でも十分な暑さだ。[p]

#雪雫
「ムカデ競争ってとても難しいわよね。[r]
中学の時にやったのだけれど、私の友達は骨折していたわ」[p]

#蛍
「骨折？！」[p]

#
そんな話を聞かされたらなおさらやりたくなくなってしまう。[p]

#雪雫
「でも雨宮くんは運動神経いいし、大丈夫だと思うわ」[p]

「ここから応援しているから」[p]

#
自分は参加できないのに、温かく声をかけてもらえるのは嬉しい。[r]

「ありがとう」[p]

;少しだけ間

#

;雪雫表示

#蛍
「どうしたの？」[p]

#雪雫
「あのね、全員リレーは出させてもらえることになったの。[r]

#蛍
「そっか、おめでとう」[p]

#雪雫
「久しぶりに走れるわ」[p]

#
やっぱり１人だけ出られないのはつまらないよな。[p]

体育の時間で、雪雫が日に当たっているところを初めて見た。[p]

#体育教師
「よーい！」[p]

;間

#
おかしい。こんな話は聞いてない。[p]

なんでオレがアンカーなんだ。[p]

#蛍
「さっき、アンカーやってって急に言われて…」[p]

#光
「日ごろさぼっているからじゃないか？」[p]

;笑いながらな感じで

#

#雪雫

「そんなことないよ」[p]

#光
「オレはこのあと部活だから、一緒に帰れないんだ。すまない」[p]

「私は水やりをしてから帰ることにするわ」[p]

「じゃあ手伝うよ」[p]

「ありがとう。でも今日は祖父のジャージ持ってきていないわ」[p]

#蛍
「この服のままやるから平気」[p]

#雪雫
「それもそうね」[p]

#光
「じゃあな」[p]

#蛍
「またね」[p]

;光の表示消す

#雪雫
「では私たちも行きましょう」[p]

;雪雫の表示消す

#
オレたちは、放課後もこうやって一緒にいることが多くなった。[p]

横断幕や応援団など、とても活気づいている。[p]

#体育教師
「よーい！」][p]

「１番だったわね。おめでとう」[p]

「ぎりぎりだったけどね」[p]

#蛍

#雪雫

雪雫はすぐ立ち上がった。[p]

しかし、先頭を走っていたが立ち上がる間に抜かされてしまって[r]

大丈夫かな。[p]

雪雫が抜かされちゃった分、オレが最後に頑張ろう。[p]

#蛍
「はぁ…はぁ…」[p]

なんとか３位から２位に順位を上げることが出来たが[r]

「大丈夫！？」[p]

#雪雫

#雪雫
「それはない、けど…」[p]

「みんなに迷惑をかけてしまったから。やっぱりでなければよかった」[p]

#蛍
「そんなことないよ。一緒に走れてよかった」[p]

#雪雫
「雨宮くんはすごいわね。カバーしてもらえてよかったわ。[r]

#雪雫
「なんだか、雨宮くん少し前と変わった気がするわ」[p]

#蛍
「そうかな。自分じゃわからない」[p]

#雪雫
「それでいいと思う。そのままでいいの」[p]

「懐かしいわね、保健室」[p]

#蛍
「ここから始まって仲良くなったみたいなものだからね」[p]

#
あの時は何となくあけていた距離も、[r]
今なら気にしないでいられる。[p]

#雪雫
「もう席に戻るわ。先に行っていてくれて大丈夫よ」[p]

#
結局赤組は総合２位だった。[r]

アンカーにされて何故か脚光を浴びてしまったし、[r]
雪雫と参加できたし、[r]
なんだかんだ悪くない体育祭になった…気がする。[p]
